Este projeto consistiu na reprodução de uma aula prática de detecção de fraudes em cartões de crédito utilizando dados históricos altamente desbalanceados. O fluxo do código envolveu a preparação dos dados por meio de transformações logarítmicas e padronização escalar (StandardScaler) com NumPy e Pandas, seguida pela criação de um modelo inicial de Regressão Logística. Para lidar com a disparidade entre transações normais e fraudulentas, foram aplicadas técnicas de amostragem (Undersampling e SMOTE) e modelos baseados em árvores, como Random Forest e XGBoost Classifier. Por fim, os hiperparâmetros foram otimizados via GridSearchCV com foco na métrica de Recall para maximizar a captura de fraudes, e a explicabilidade das decisões do modelo foi analisada graficamente por meio da biblioteca SHAP.
